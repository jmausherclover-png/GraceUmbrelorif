# Grace Umbrelorif

This is the README for the Grace Umbrelorif pharmacy & clinic management system.

## Features
- Geofence attendance
- HR management
- Payroll modules

## Installation

```bash
# Clone the repository
$ git clone https://github.com/jmausherclover-png/GraceUmbrelorif.git

# Move into the directory
$ cd GraceUmbrelorif

# Install dependencies
$ npm install
```

## Usage

```bash
# Start the application
$ npm start
```
